## Failles & Sécurisation

- JWT : un jeton sécurisé qui identifie un utilisateur et ses droits
- SQL Injection : voler ou modifier une base de données
- CORS : contrôle quels sites peuvent accéder à mon API
- Chiffrement / Hashage : protège correctement le mot de passe grâce au hashage,mais faisons le point en détail
